# WeCure Pharmaceuticals

## Overview

WeCure Pharmaceuticals is a C++ console-based pharmacy management system designed to manage medicine records efficiently. The system helps in maintaining pharmacy inventory, handling medicine information, and generating bills. It simplifies day-to-day pharmacy operations while reducing manual effort and improving accuracy.

## Features

* Add new medicine records
* Update existing medicine details
* Search medicines by name or ID
* Delete medicine records
* Display all available medicines
* Generate customer bills
* Manage medicine inventory efficiently

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* File Handling

## Objectives

* Automate pharmacy inventory management
* Reduce manual record-keeping
* Improve data accuracy and accessibility
* Simplify billing and medicine tracking

## System Modules

### Medicine Management

* Add medicines
* Update medicine information
* Delete medicines
* Search medicines

### Inventory Display

* View all available medicines
* Check medicine details

### Billing System

* Generate bills for purchased medicines
* Calculate total cost automatically

## Benefits

* User-friendly console interface
* Fast and efficient medicine record management
* Reduced paperwork and human errors
* Easy maintenance and scalability

## Future Enhancements

* Graphical User Interface (GUI)

## Conclusion

WeCure Pharmaceuticals provides a simple and effective solution for pharmacy management using C++. The project demonstrates the practical application of programming concepts such as object-oriented programming and file handling to solve real-world inventory and billing challenges.
